# FYP Trekking Platform Prototype

This repository contains a full-stack FYP prototype for a trekking and expedition booking platform based around Everest Encounter Treks.

## Current Scope

### Frontend

- Home page
- Trek listing page
- Trek details page
- Login page
- Register page
- Guide page
- Gallery page
- Language selection
- Currency selection

### Backend

- Authentication module
- Region module
- Trek module
- Booking module
- Review module
- Inquiry module
- Gallery module
- Hero content module
- Payment record model
- Health endpoints

## Tech Stack

- Client: React, Vite, Tailwind CSS, Redux Toolkit, RTK Query
- Server: Node.js, Express, MongoDB, Mongoose, JWT, Multer

## Structure

- `client/`: React frontend
- `server/`: Express backend

## Notes

- This is a prototype and not a finished production deployment.
- Some backend capability is broader than the currently exposed public client routes.
- Environment files and secrets are intentionally excluded from version control.
