# FYP Auth Module (Login + Signup)

This repository contains only the authentication-related implementation extracted from the original FYP project:
- Frontend login page
- Frontend signup/register page
- Frontend auth API integration and auth state slice
- Backend auth routes/controller/service
- Backend user model and JWT helpers used by auth

Notes:
- `.env` and secrets are intentionally excluded.
- `node_modules` and uploads are excluded.
